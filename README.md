📊 Sizy – Disk Usage Analyzer

Find what’s taking up space on your PC in seconds.

🚀 What is Sizy?

Sizy is a lightweight Python tool that scans a folder and shows the largest files and directories inside it.

It helps you quickly understand what is using your disk space.

✨ Features
📁 Shows top 10 largest files and folders
🔁 Scans directories recursively
📏 Human-readable sizes (B, KB, MB, GB, TB)
🔒 Skips files you don’t have permission to access
⚡ Very lightweight and fast
🖥️ Requirements
Python 3.8 or higher

Download Python:
https://www.python.org/downloads/

Make sure to enable:
✔ “Add Python to PATH”

▶️ How to use
Download sizy.py
Open terminal / CMD
Navigate to the file location
Run:
python sizy.py
📌 Example

Sizy will scan a folder and display something like:

Largest file: 1.2 GB
Biggest folder: 4.5 GB
⚠️ Notes
Read-only tool (does NOT modify files)
Works on Windows
Requires Python installed
May skip protected system files
💡 Why Sizy?

Your disk gets full and you don’t know why?
Sizy shows you exactly what’s using space.
